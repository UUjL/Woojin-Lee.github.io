Notaker

AI-Powered Notes from Audio & Transcripts

Sep 2025 — Apr 2026

Python FastAPI Django Electron React Next.js

Overview

Notaker is a local-first AI note-taking service that generates structured notes from recorded audio, live audio, and transcripts. The product ships as both a desktop app and a web app.

Local backend (FastAPI)

A Python FastAPI backend powers the desktop experience on the user's machine. It exposes unified APIs for the Electron client, including:

  • Speech-to-text transcription with Whisper
  • File and SQLite synchronization
  • Attachment and image handling
  • Translation
  • Embedding-based search and RAG pipelines

Cloud integration (Django)

The system connects to a remote Django server with JWT-based authentication. Proxy routing was implemented so the Electron (React) client and backend services can share tokens seamlessly across local and remote services.

Stack

Desktop: Electron, React
Web: Next.js, React
Local services: Python, FastAPI, SQLite, Whisper, RAG
Remote services: Django, JWT auth
